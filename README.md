# ChatPulse

ChatPulse — Telegram-бот для аналітики та гейміфікації груп. Він рахує активність, порівнює тижні, видає XP, рівні й досягнення, підтримує серії активності та автоматично надсилає красиві тижневі картки.

## Приватність

Бот **не зберігає тексти повідомлень, підписи або файли**. У базі залишаються Telegram ID, ID повідомлення, автор, часові мітки, числові лічильники та поточна кількість реакцій.

Для антиспаму зберігаються лише keyed fingerprints: точний відбиток і 64-бітний similarity fingerprint. Вони створюються із секретним ключем сервера, тому оригінальний текст у базу не записується.

## Можливості

- статистика за сьогодні, 7 днів, місяць і весь час;
- `/compare` — поточні 7 днів проти попередніх 7 днів;
- топ учасників та особисті профілі;
- повідомлення тижня з кнопкою переходу;
- груповий і глобальний рівні ChatPulse;
- досягнення з комбінованими сповіщеннями;
- streak із трьома захисними днями на календарний місяць;
- автоматичні PNG-картки в трьох темах;
- точний час звіту у форматі `HH:MM`;
- PostgreSQL/Supabase, FastAPI webhook і Google Cloud Run.

## Команди

| Команда | Дія |
|---|---|
| `/start` | Реєстрація та кнопка додавання бота до групи |
| `/help` | Коротка інструкція |
| `/stats` | Статистика з кнопками вибору періоду |
| `/today`, `/week`, `/month`, `/all` | Швидкий вибір періоду |
| `/top` | Рейтинг учасників |
| `/me` | Базова особиста статистика |
| `/profile` | XP, груповий/глобальний рівень, streak і досягнення |
| `/compare` | Порівняння двох останніх 7-денних періодів |
| `/weekly` | PNG-картка тижневого звіту |
| `/settings` | Налаштування групи для адміністратора |
| `/setreporttime 20:30` | Точний локальний час звіту |
| `/resetstats` | Підтверджене скидання статистики групи |

## XP та антиспам

Базові нагороди:

- повноцінне текстове повідомлення: `+1 XP`;
- відповідь: `+2 XP`;
- фото або голосове/video note: `+2 XP`;
- отримана реакція: `+3 XP`.

Захист від накрутки:

- XP не частіше ніж раз на 5 секунд;
- повідомлення з 1–2 символів, emoji-only та стікери не дають XP;
- точні та майже однакові повідомлення блокуються;
- після 20 повідомлень за 10 хвилин XP зменшується;
- після 40 повідомлень за 10 хвилин повідомлення тимчасово не дають XP;
- максимум `200 XP` на день у кожній групі;
- максимум `400 XP` на день у глобальний профіль.

Звичайні лічильники активності продовжують працювати після досягнення XP-ліміту. Рівень `L` починається на сумарному XP `50 × (L - 1) × L`. Тири: Новачок, Бронза, Срібло, Золото, Діамант.

## Серії та досягнення

День зараховується в streak після щонайменше `10 XP` у групі. На кожен календарний місяць доступні три автоматичні захисні дні; вони не переносяться далі.

Важливі досягнення та підвищення рівня бот оголошує одразу. Менші досягнення видно у `/profile` та тижневому звіті. Одне досягнення не видається повторно.

## Теми картки

Адміністратор перемикає тему в `/settings`:

- `Dark Pulse`;
- `Telegram Wave`;
- `Clean Light`.

## Локальний запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
cp .env.example .env
uvicorn app.main:create_app --factory --reload --port 8080
```

Основні змінні:

```env
BOT_TOKEN=123456:your-real-token
WEBHOOK_BASE_URL=https://your-service.run.app
WEBHOOK_PATH_SECRET=long-random-path-secret
WEBHOOK_HEADER_SECRET=long-random-header-secret
SCHEDULER_SECRET=long-random-scheduler-secret
DATABASE_URL=postgresql://user:password@pooler-host:5432/postgres
DEFAULT_TIMEZONE=Europe/Kyiv
```

## Автоматичні щотижневі звіти

Захищений endpoint:

```text
POST /internal/weekly-reports
X-ChatPulse-Scheduler-Secret: <SCHEDULER_SECRET>
```

Через кастомні хвилини Cloud Scheduler має запускати перевірку кожні 5 хвилин:

```text
URL: https://your-service.run.app/internal/weekly-reports
Method: POST
Header: X-ChatPulse-Scheduler-Secret=<SCHEDULER_SECRET>
Schedule: */5 * * * *
Timezone: Etc/UTC
```

Оновлення наявного job:

```bash
gcloud scheduler jobs update http chatpulse-weekly-reports \
  --location=europe-central2 \
  --schedule="*/5 * * * *" \
  --time-zone="Etc/UTC"
```

Endpoint сам перевіряє часовий пояс, день, годину й хвилину кожної групи та не надсилає повторний звіт того самого локального дня.
